Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are:
1. getElementById()
2. getElementByClassName()
3. getElementByTagName()
4. querySelector()
5. querySelectorAll()
6. getElementByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. InnerHTML: it is used to get or set the HTML content inside an element including tags.
2. innerText: it is used to get or set the text content inside an element. Like input area.
3. textContent: it is similar to innerText. it sets the text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation refers to the process of using event propagation to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event, like a click or a keypress, starts at the most specific target.

Question-5: How do you create, add, and remove elements using JavaScript?

1. create element: document.createElement()
2. add element: appendChild()
3. remove element: remove() or parentNode.removeChild()
